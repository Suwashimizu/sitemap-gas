# sitemap-gas
GASでサイトマップを作成します（したい）

以下のURLからコピーしてください
https://docs.google.com/spreadsheets/d/1-i6dpTrWCTsc7BImV_Gi_hAxEbZ6Ku8W7Qg0BTO4yC8/copy

# Using

B2のセルに取得したいサイトを入力してください。
入力後メニューのサイトマップ>サイトマップを取得を実行後
シート上にリンク一覧が書き込まれます。

なお実行時にシートは一度空になるため
続けて取得したい場合はシートをコピーするなどしてください。
